Understanding the Project

A deep dive into the architecture, core technologies, and goals behind the Sentiment Analyzer.

Architecture Overview

This project demonstrates the powerful integration of a **dynamic frontend** with a robust **Java Spring Boot backend**. The primary goal is to showcase the efficient packaging and deployment of a machine learning artifact. The backend exposes a clean RESTful API (`POST /analyze`) that accepts JSON text input and returns precise sentiment analysis results.
